JavaScript 介绍 ​

基本语法 ​

变量声明 ​

JavaScript 提供了三种变量声明方式：

  • var: 函数作用域，存在变量提升。
  • let: 块级作用域，不存在变量提升。
  • const: 块级作用域，用于声明常量。
javascript
var name = 'Alice'
let age = 25
const PI = 3.14

数据类型 ​

JavaScript 有以下基本数据类型：

  • Number: 数字
  • String: 字符串
  • Boolean: 布尔值
  • Null: 空值
  • Undefined: 未定义
  • Symbol: 唯一标识符
  • Object: 对象

运算符 ​

  • 算术运算符 (+, -, *, /)
  • 比较运算符 (==, ===, !=, !==)
  • 逻辑运算符 (&&, ||, !)

控制结构 ​

  • 条件语句 (if, else, switch)
  • 循环 (for, while, do...while)
javascript
if (age >= 18) {
  console.log('成年人')
} else {
  console.log('未成年人')
}

函数 ​

函数是 JavaScript 的一等公民，可以定义和调用。

javascript
function greet(name) {
  return `Hello, ${name}!`
}
console.log(greet('Alice')) // 输出: Hello, Alice!

闭包 ​

定义 ​

闭包是指函数能够访问其词法作用域之外的变量，即使函数在其词法作用域之外执行。

作用 ​

  • 封装私有变量。
  • 实现函数工厂。

常见用例 ​

javascript
function createCounter() {
  let count = 0
  return function () {
    return ++count
  }
}

const counter = createCounter()
console.log(counter()) // 输出: 1
console.log(counter()) // 输出: 2

注意事项 ​

  • 闭包可能导致内存泄漏，因为外部变量不会被垃圾回收。
  • 避免滥用闭包。
