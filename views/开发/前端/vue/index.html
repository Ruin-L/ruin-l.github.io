Vue 介绍 ​

Vue.js（通常简称为 Vue）是一个用于构建用户界面的 渐进式 JavaScript 框架 ，由前 Google 工程师尤雨溪（Evan You）于 2014 年创建。它以其简洁性、灵活性和高性能著称，已成为前端开发的主流工具之一。以下是 Vue 的核心特点和技术亮点：

核心特性 ​

  1. 渐进式框架
    • Vue 可逐步集成到项目中：可作为轻量库嵌入部分页面，也可构建复杂的单页应用（SPA）。
    • 按需引入功能（如路由、状态管理），不强制全家桶。
  2. 响应式数据绑定
    • 通过 Object.defineProperty（Vue 2）或 Proxy（Vue 3）实现 数据驱动视图 。
    • 数据变化时，视图自动更新，无需手动操作 DOM。
  3. 组件化开发
    • 将 UI 拆分为独立可复用的组件（.vue 文件包含模板、逻辑、样式）。
    • 支持父子组件通信（props/$emit）、依赖注入（provide/inject）等。
  4. 虚拟 DOM
    • 高效对比变化并局部更新真实 DOM，提升渲染性能。

核心语法示例 ​

html

js
<div id="app">
  <p>{{ message }}</p>         <!-- 文本插值 -->
  <button @click="reverseText">反转</button> <!-- 事件绑定 -->

  const { createApp, ref } = Vue;
  createApp({
    setup() {
      const message = ref('Hello Vue!'); // 响应式数据
      const reverseText = () => {
        message.value = message.value.split('').reverse().join('');
      };
      return { message, reverseText };
    }
  }).mount('#app');
</script>

Vue 3 重大革新（2020 年发布） ​

  1. Composition API
    • 替代 Options API，通过 setup() 函数实现逻辑复用（类似 React Hooks）。
    • 代码按功能组织，而非分散在 data/methods 等选项中。
  2. 性能优化
    • 重写虚拟 DOM（Diff 算法优化）。
    • 支持 Tree-shaking，减少打包体积。
  3. TypeScript 深度集成
    • 源码使用 TypeScript 重写，提供完善的类型支持。
  4. 新特性
    • Teleport：将组件渲染到 DOM 任意位置。
    • Suspense：异步组件加载状态管理。
    • 多个根节点支持（Fragment）。

适用场景 ​

  • 快速构建动态 Web 应用或复杂 SPA。
  • 需高开发效率的中小型项目（对比 React/Angular 更易上手）。
  • 渐进式升级现有项目（可替代 jQuery）。

注意点 ！ ​

在 Vue 2 中，直接通过索引修改数组的值（如 array[index] = newValue） 不会触发响应式更新 ，这是因为 Vue 2 的响应式系统基于 Object.defineProperty，它无法检测数组索引的直接赋值操作。

🔧 解决方案：正确修改数组值 ​

以下是触发响应式更新的三种方法：

方法 1：使用 Vue.set() 或 this.$set() ​

javascript

// 全局方法
Vue.set(yourArray, index, newValue);

// 组件内方法
this.$set(this.yourArray, index, newValue);

⚠️ 为什么需要这样处理？ ​

Vue 2 的响应式限制：

  1. 数组索引修改 ：Object.defineProperty 无法监听数组索引变化
  2. 长度修改 ：直接修改 length 属性也不触发更新
  3. 解决方式 ：Vue 重写了数组的 7 个变更方法（push/pop/shift/unshift/splice/sort/reverse）

💡 最佳实践建议 ​

  1. 优先使用 $set 或 splice 简单直观，Vue 官方推荐方式

  2. 复杂数据结构考虑 Vue 3 Vue 3 使用 Proxy 可完美解决数组索引更新问题

  3. 避免直接修改深层数组 嵌套数组时同样需要特殊处理： javascript

    js
    // 修改嵌套数组
this.$set(this.data[0].children, 1, newValue)

🌰 完整示例 ​

vue

js
<template>
  <div>
    <ul>
      <li v-for="(item, index) in items" :key="index">
        {{ item }}
        <button @click="updateItem(index)">更新</button>
      </li>
    </ul>
  </div>
</template>

<script>
export default {
  data() {
    return {
      items: ['A', 'B', 'C']
    }
  },
  methods: {
    updateItem(index) {
      // ✅ 正确方法 1
      this.$set(this.items, index, `Updated ${index}`);

      // ✅ 正确方法 2
      // this.items.splice(index, 1, `Updated ${index}`);

      // ❌ 错误方法（不会触发更新）
      // this.items[index] = `Updated ${index}`;
    }
  }
}
</script>

⚡ 性能提示 ​

当需要处理大型数组时：

  • 优先使用 splice 避免创建新数组
  • 避免在模板中使用复杂表达式
  • 必要时使用虚拟滚动（如 vue-virtual-scroller）

关键总结 ：在 Vue 2 中修改数组时，永远避免直接通过索引赋值，始终使用 $set、splice 或返回新数组的方式操作数据。这是 Vue 2 响应式系统的设计限制，在 Vue 3 中已通过 Proxy API 完美解决。
