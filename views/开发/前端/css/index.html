CSS 开发指南 ​

1. CSS 简介 ​

CSS（层叠样式表）是一种用于描述网页外观和格式的语言。它与 HTML 和 JavaScript 一起构成了现代网页开发的三大核心技术。CSS 可以控制网页的布局、颜色、字体等样式，使网页更加美观和易于维护。

2. CSS 布局方式 ​

2.1 Flex 布局 ​

Flex 布局是一种一维布局模型，适合在行或列方向上排列元素。

基本属性 ​

  • display: flex;：将容器设置为 Flex 布局。
  • flex-direction：定义主轴方向（row、column、row-reverse、column-reverse）。
  • justify-content：定义主轴对齐方式（flex-start、flex-end、center、space-between、space-around）。
  • align-items：定义交叉轴对齐方式（flex-start、flex-end、center、baseline、stretch）。

示例代码 ​

html
<div class="flex-container">
  <div class="flex-item">Item 1</div>
  <div class="flex-item">Item 2</div>
  <div class="flex-item">Item 3</div>
</div>

<style>
  .flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    height: 200px;
    background: #f0f0f0;
  }
  .flex-item {
    width: 100px;
    height: 100px;
    background: #007bff;
    color: white;
    text-align: center;
    line-height: 100px;
  }
</style>

2.2 Grid 布局 ​

Grid 布局是一种二维布局模型，适合复杂的网格布局。

基本属性 ​

  • display: grid;：将容器设置为 Grid 布局。
  • grid-template-columns：定义列宽。
  • grid-template-rows：定义行高。
  • grid-gap：定义网格间距。

示例代码 ​

html
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
</div>

<style>
  .grid-container {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    grid-gap: 10px;
    padding: 10px;
    background: #f0f0f0;
  }
  .grid-item {
    background: #007bff;
    color: white;
    text-align: center;
    padding: 20px;
  }
</style>

4. REM 适配大屏 ​

REM（Root EM）是基于根元素（<html>）的字体大小的单位。通过动态调整根元素的字体大小，可以实现适配不同屏幕尺寸。

实现方式 ​

javascript
// 动态计算 REM
function setRem() {
  const docEl = document.documentElement
  const width = docEl.clientWidth
  const rem = width / 10 // 假设设计稿宽度为 750px，则 1rem = 75px
  docEl.style.fontSize = rem + 'px'
}
window.addEventListener('resize', setRem)

5. CSS 变量 ​

CSS 变量（自定义属性）允许在 CSS 中定义可重用的值。

定义和使用 ​

css
:root {
  --primary-color: #007bff;
  --secondary-color: #6c757d;
}

.button {
  background-color: var(--primary-color);
  color: white;
  padding: 10px 20px;
}

6. CSS 预处理器 ​

6.1 SCSS ​

SCSS 是 Sass 的一种语法扩展，支持变量、嵌套、混合等功能。

安装 ​

bash
npm install sass --save-dev
1

示例代码 ​

scss
$primary-color: #007bff;

.button {
  background-color: $primary-color;
  color: white;
  padding: 10px 20px;

  &:hover {
    background-color: darken($primary-color, 10%);
  }
}
