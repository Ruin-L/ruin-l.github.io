TypeScript 介绍 ​

TypeScript 是由微软开发的开源编程语言，它是 JavaScript 的超集（即所有合法的 JavaScript 代码都是合法的 TypeScript 代码），通过添加 静态类型系统 和 面向对象编程特性 来提升代码的可维护性和开发效率。TypeScript 最终会被编译为纯 JavaScript，因此可以运行在任何支持 JavaScript 的环境中。

核心特性详解 ​

1. 静态类型系统（Type System） ​

  • 类型注解：显式声明变量、函数参数和返回值的类型。
    typescript
    let name: string = 'Alice'
function add(a: number, b: number): number {
  return a + b
}

  • 高级类型：
    • 联合类型（Union Types）：string | number
    • 交叉类型（Intersection Types）：type A = { a: string } & { b: number }
    • 类型保护（Type Guards）：通过 typeof、instanceof 或自定义谓词（is 关键字）缩小类型范围。
      typescript
      function isString(value: unknown): value is string {

6. 高级工具类型 ​

  • Partial<T>：将类型所有属性变为可选。
  • Required<T>：将类型所有属性变为必填。
  • Pick<T, K>：从类型 T 中选择部分属性 K。
  • Record<K, T>：构造一个类型，其属性键为 K，值为 T。

优势与适用场景 ​

  • 优势：
    • 在开发阶段捕获类型错误，减少运行时错误。
    • 提供强大的 IDE 支持（如 VS Code 的智能提示、重构工具）。
    • 支持大型项目的模块化开发和团队协作。
  • 适用场景：
    • 企业级大型应用（如 Angular、Vue 3、React 项目）。
    • 需要长期维护的代码库。
    • 对类型安全要求严格的场景（如金融、医疗系统）。

简单示例 ​

typescript
// 定义接口
interface Book {
  title: string
  price: number
}

// 泛型函数
function getProperty<T, K extends keyof T>(obj: T, key: K): T[K] {
  return obj[key]
}

const book: Book = { title: 'TypeScript Guide', price: 29.99 }
console.log(getProperty(book, 'title')) // 输出 "TypeScript Guide"
