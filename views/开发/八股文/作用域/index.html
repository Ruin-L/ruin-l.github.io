作用域链 ​

JavaScript 的作用域链（Scope Chain）是变量和函数查找机制的核心，它决定了代码中哪些数据能被访问。

查找规则 ​

  • 当访问变量时，引擎从当前作用域开始查找，若未找到则沿作用域链向上逐级查找，直到全局作用域。
  • 若全局作用域仍未找到，则返回 undefined（非严格模式）或抛出 ReferenceError。

示例分析 ​

javascript
let globalVar = 'global'
function outer() {
  let outerVar = 'outer'
  function inner() {
    console.log(globalVar) // 查找全局变量
    console.log(outerVar) // 查找外部函数变量
  }
  inner()
}
outer() // 输出: "global" 和 "outer"
