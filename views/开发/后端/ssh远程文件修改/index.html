ssh 远程文件修改 ​

建议通过 vscode 的插件Save as Root in Remote - SSH，进行远程服务器文件的修改，不然的话每次都要使用 vim 并执行 sudo 才能修改成功。

使用方法 ​

ctrl + p 打开搜索框，之后输入**>save**，即可看到保存

对于新增新文件可以右键你想要新增文件的文件夹。
